## GatewayClient

```python
from minions_openclaw import GatewayClient
import asyncio

async def main():
    client = GatewayClient('ws://localhost:8080', token='token')
    await client.open_connection()
    presence = await client.fetch_presence()
    await client.close()

asyncio.run(main())
```
