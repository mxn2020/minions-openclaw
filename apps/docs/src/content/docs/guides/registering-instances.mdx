---
title: Registering Instances
description: How to register OpenClaw Gateway instances
---

## Register via CLI

```bash
openclaw-manager register <name> --url <websocket-url> [--token <auth-token>]
```

## Register via TypeScript SDK

```typescript
import { InstanceManager } from '@minions-openclaw/core';
const manager = new InstanceManager();
const instance = await manager.register('my-gateway', 'ws://localhost:8080', 'token');
```
