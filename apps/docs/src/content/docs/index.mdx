## What is Minions OpenClaw?

Minions OpenClaw is a production-quality SDK and CLI for managing OpenClaw Gateway instances. It uses [minions-sdk](https://www.npmjs.com/package/minions-sdk) for structured data management.

## Quick Start

```bash
npm install -g @minions-openclaw/cli
openclaw-manager register my-gateway --url ws://localhost:8080
openclaw-manager list
openclaw-manager snapshot <instanceId>
```
